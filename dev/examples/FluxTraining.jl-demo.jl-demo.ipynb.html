FluxTraining.jl demo

This is a quick-and-dirty demo for FluxTraining.jl that shows that it’s already possible to easily and quickly train a large convolutional network on a dataset bigger than MNIST.

(training on a GTX 1080 ti)

Note, however, that large parts of the interfaces of the packages are unclean and unstable.

Setup

Install packages with


] add https://github.com/lorenzoh/DataAugmentation.jl#master

] add https://github.com/lorenzoh/FluxTraining.jl#master

] add https://github.com/lorenzoh/VisionDatasets.jl#master

] add Flux, Images


using DataAugmentation
using FluxTraining
using VisionDatasets
using Images
using Flux

Load data (from VisionDatasets.jl)

# Load the ImageNette dataset and split it into training and validation
dataset = ImageNette.ImageNette2_320();
trainds, valds = FluxTraining.splitdataset(dataset, dataset.data[:split] .== :train);

Data transformation pipeline

# Returns a dict with keys :image and :label
rawsample = getobs(trainds, 1)

# Convert the dictionary to a tuple of (x, y)
toitem(rawsample::Dict) = (Image(rawsample[:image]), Label(rawsample[:label]))
toxy((itemx, itemy)::Tuple) = (itemdata(itemx), itemdata(itemy))
toxy (generic function with 1 method)
# precomputed datasets to normalize inputs
MEANS, STDS = (
    [0.4624031671459694, 0.458004282833669, 0.42971805872627555],
    [0.23547507202366247, 0.22990946844849633, 0.23796827581243238]);
xtfm_train = RandomResizedCrop((224, 224)) |> ToEltype(RGB) |> ToTensor() |> Normalize(MEANS, STDS)
xtfm_val = CenterResizedCrop((224, 224)) |> ToEltype(RGB) |> ToTensor() |> Normalize(MEANS, STDS)
    
ytfm = OneHot(10)
    
tfm_train = toxy ∘ XYPipeline(xtfm_train, ytfm) ∘ toitem
tfm_val = toxy ∘ XYPipeline(xtfm_val, ytfm) ∘ toitem;

Transform data

databunch = DataBunch(
    DataLoader(trainds, 32; transformfn = tfm_train, numworkers = 12),
    DataLoader(valds, 64; transformfn = tfm_val, numworkers = 12),
);
model = Chain(Models.xresnet18(), Models.classificationhead(10, 512)) |> gpu;
learner = Learner(
    model,
    databunch,
    Flux.ADAM(),
    Flux.crossentropy;
    callbacks = [ProgressBarLogger(), MetricsLogger()],
    metrics = [Accuracy()],
    schedule = onecycleschedule(10, 0.01));
fit!(learner, 10);
