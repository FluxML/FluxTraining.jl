FluxTraining.LogHyperParams

LogHyperParams(backends...) <: Callback

Callback that logs hyperparameters to one or more FluxTraining.LoggerBackends.

See also FluxTraining.LoggerBackend, FluxTraining.Loggables.Loggable, FluxTraining.log_to, TensorBoardBackend

Example

logcb = LogHyperParams(TensorBoardBackend("tblogs"))
schedule = ...
Learner(model, data, opt, lossfn, Scheduler(LearningRate => schedule), logcb)
