FluxTraining.epoch!

epoch!(learner, phase[, dataiter])

Train learner for one epoch on dataiter. Iterates through dataiter and step!s for each batch/item. If no data iterator is passed in, use learner.data[phasedataiter(phase)].

This behavior can be overloaded using a custom FluxTraining.Phases.Phase.
