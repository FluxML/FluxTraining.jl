FluxTraining.SanityCheck

SanityCheck([checks; usedefault = true])

Callback that runs sanity FluxTraining.Checks when the Learner is initialized. If usedefault is true, it will run all checks in FluxTraining.CHECKS in addition to the ones you pass in.
